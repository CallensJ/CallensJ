- **Framework** : Next.js 15 (App Router)
- **Langage** : TypeScript
- **Styling** : Tailwind CSS v4)
- **Animations** : Framer Motion
- **Déploiement** : Vercel

- SSR + optimisation SEO (metadata dynamiques, Schema.org, sitemap)
- Design system custom (tokens couleurs, typographie Playfair Display + Poppins)
- Animations scroll-triggered avec Framer Motion
- Pages : Accueil, Prestations, Réalisations, Contact, Curriculum
- Responsive mobile-first
- Score PageSpeed 90+

# Erudia — Plateforme éducative pour enfants

**[Erudia](https://erudia.app)** est une application web éducative bilingue (FR/EN) destinée aux enfants de 6 à 11 ans.  
Quiz interactifs, progression gamifiée, espace parent et abonnement intégré.

## 🚀 Stack

- **Framework** : Next.js 15 (App Router)
- **Langage** : TypeScript
- **Styling** : Sass 7-1 
- **Auth** : NextAuth.js
- **Paiement** : Stripe (abonnements récurrents)
- **Déploiement** : Vercel

## ✨ Features

- Authentification complète (inscription, connexion, gestion de compte)
- Abonnements via Stripe (mensuel / annuel)
- Interface bilingue FR / EN
- Quiz adaptatifs par niveau et matière , 20 categories , Methodologie FLE ( Francais langue Etrangere)
- Espace parent et suivi de progression enfant
- PWA-ready

## 🌐 Live

- Landing : [erudia.app](https://erudia.app/fr)
- Application : [app.erudia.app](https://app.erudia.app/fr/)

# Kidoplan — Organisateur de tâches pour enfants

**[Kidoplan](https://kidoplan.vercel.app)** est une Progressive Web App (PWA) destinée aux enfants,  
permettant de visualiser et organiser leurs tâches quotidiennes et récurrentes sur une semaine.

## 🚀 Stack

- **Framework** : Vue.js 3 (Composition API)
- **Base de données** : Supabase (PostgreSQL + Auth)
- **Styling** : SASS 7 - 1
- **Déploiement** : Vercel

## ✨ Features

- Vue hebdomadaire des tâches (7 jours)
- Tâches récurrentes configurables
- Interface pensée pour les enfants (UX simplifiée, visuels clairs)
- PWA installable sur mobile et desktop
- Authentification via Supabase Auth
- Données persistantes en temps réel

## 🌐 Live

[kidoplan.vercel.app](https://kidoplan.vercel.app)
